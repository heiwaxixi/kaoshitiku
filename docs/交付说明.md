<span style="color:#0f8f69;font-weight:700">【新增】</span> 已实现功能：

- 题库按科目、难度、题型、关键词筛选。
- 单选、多选题作答与即时判分。
- 错题自动进入优先复盘队列。
- 本地错题教练输出错因、标准答案、陷阱和强化动作。
- 掌握度统计按科目展示进度。
- 今日强化路径支持把题目加入计划并跳转练习。
- 桌面三栏布局与移动端纵向布局均已截图验证。
- <span style="color:#2563eb;font-weight:700">【新增-导入题库】</span> 支持粘贴题块文本、上传 TXT/CSV/XLSX，并自动识别题型、题干、选项、答案和解析。
- <span style="color:#7c3aed;font-weight:700">【新增-题库目录】</span> 各题库以目录卡片单独展示，点击目录后进入独立练习界面。
- <span style="color:#7c3aed;font-weight:700">【改动-导入建目录】</span> 导入确认后不再并入当前题库，而是自动建立新的题库目录并进入该目录练习。
- <span style="color:#7c3aed;font-weight:700">【新增-折叠面板】</span> 本地 AI 教练、错题队列、学习画像、今日计划支持折叠摘要和展开详情。
- <span style="color:#d97706;font-weight:700">【新增-练习页布局】</span> 当前题目拉宽为上方主练习区，本地 AI 教练、错题队列、学习画像、今日计划移动到下方同一网格并对齐显示。
- <span style="color:#d97706;font-weight:700">【新增-显示答案】</span> 题目操作区新增“显示答案”按钮，点击后正确选项以绿色高亮，标准答案和答案解析在题目下方突出展示。
- <span style="color:#0891b2;font-weight:700">【新增-题库清单折叠】</span> 当前题库清单支持折叠摘要和展开详情，默认显示当前筛选题目数量，展开后仍可点击题目跳转练习。
- <span style="color:#db2777;font-weight:700">【改动-答案持续显示】</span> “显示答案”改为持续开关，点击下一题或切换题目不再自动关闭，需要手动点击“隐藏答案”取消。
- <span style="color:#16a34a;font-weight:700">【新增-上一题】</span> 题目操作区新增“上一题”按钮，可在当前筛选结果内向前切换题目，并保持显示答案等当前练习状态。
- <span style="color:#e11d48;font-weight:700">【改动-取消访问码】</span> 已取消内测访问码入口，打开 GitHub Pages 地址后直接进入题库目录；手机不需要同一 WiFi 即可测试。
- <span style="color:#0ea5e9;font-weight:700">【新增-PWA】</span> 已加入 Web App manifest、service worker 和应用图标，支持 iPhone Safari “添加到主屏幕”形成类 App 入口。
- <span style="color:#0ea5e9;font-weight:700">【新增-启动脚本】</span> 新增 `启动公网内测.bat`，用于重新启动生产预览和 Cloudflare quick Tunnel；临时公网 URL 每次可能变化。
